RhinoSpatial

Spatial Context Helper

Draw one rectangle on the map. The Spatial Context component in Grasshopper should update automatically. You do not need to work with the raw bounding box numbers for normal use.

How To Use It

1. Click the rectangle tool.

2. Drag the area you want.

3. Go back to Grasshopper. Spatial Context should update by itself.

If Reference Service and Reference Layer Name are connected, this map will try to zoom to the selected layer automatically.
